## 이중 연결 리스트

모든 노드가 이전 노드와 다음 노드에 대한 정보를 모두 저장하고 있는 연결 리스트. 두 개의 노드 링크를 사용하므로 어느 방향으로든 리스트를 순회할 수 있다.

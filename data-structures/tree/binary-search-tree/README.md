## 이진 탐색 트리

이진 탐색 트리는 이진트리 기반의 탐색을 위한 자료구조이다. 이진 탐색의 효율적인 탐색 능력을 가지며, 삽입과 삭제가 가능한 것이 특징이다.

## 이진 탐색 트리의 4가지 조건

- 모든 노드의 키는 유일하다.
- 왼쪽 서브 트리의 키들은 루트의 키보다 작다.
- 오른쪽 서브 트리의 키들은 루트의 키보다 크다.
- 왼쪽과 오른쪽 서브 트리도 이진 탐색 트리이다.

## 복잡도

### 시간 복잡도

시간 복잡도(Time Complexity) : 알고리즘의 수행시간 분석 결과

| 접근 | 탐색 | 삽입 | 삭제 |
| :--: | :--: | :--: | :--: |
| O(n) | O(n) | O(1) | O(1) |

### 공간 복잡도

공간 복잡도(Spcae Complexity) : 알고리즘의 메모리 사용량에 대한 분석 결과

O(n)

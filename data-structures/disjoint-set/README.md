## 서로소 집합

서로소 집합이란 서로 중복되지 않는 부분 집합들로 나눠진 원소들에 대한 정보를 저장하고 조작하는 자료 구조이다. 전체 집합이 있을 때 구성 원소들이 겹치지 않도록 분할하는 데 자주 쓰인다
